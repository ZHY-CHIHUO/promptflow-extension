# 📋 PromptFlow

## 🎯 是什么

PromptFlow 是一个 Chrome 浏览器扩展。在 ChatGPT、Claude、Gemini 或任何 AI 对话页面上，一个紫色悬浮球始终悬浮在页面上。单击打开面板，一键保存当前输入框的提示词。需要时，从面板中搜索、编辑、回填。提示词可以存为带变量的模板，填入页面时按需替换。

核心思想很朴素：**你精心打磨过的提示词，不应该在输入框里用完就丢。**

---

## ✨ 功能

| 功能 | 说明 |
|------|------|
| **💾 一键保存** | 点击悬浮球 → 保存，当前 AI 输入框的内容即刻入库 |
| **📋 提示词库** | 搜索、编辑、删除、回填，所有提示词按时间排序 |
| **📝 模板系统** | 把提示词存为模板，用 `{{变量名}}` 表示变量，填入时按需替换 |
| **✨ 一键优化** | 自动填入优化提示词的元提示词，帮你生成更好的版本 |
| **📥 提取 AI 回复** | 从对话页面自动抓取 AI 回复，保存为提示词或模板 |
| **🎨 变量填充弹窗** | 模板有变量时，弹出居中毛玻璃弹窗，左侧实时预览，右侧逐项填写 |
| **☁️ 云端同步** | 登录 Chrome 账号后自动跨设备同步，无需注册 |
| **📤 导入/导出** | JSON 格式的备份文件，安心迁移或分享 |
| **⚙️ 设置** | 开关悬浮球、网站黑名单（不显示悬浮球）、清空数据 |
| **🔒 隐私优先** | 所有数据存本地 IndexedDB，不上传任何服务器 |
